# Contributing to BharatAI

We welcome contributions to BharatAI! Here are some guidelines to help you get started:

## Contributing Guidelines
- **Fork the repository**: Start by forking the repository to your own account.
- **Create a new branch**: Use a descriptive name for your branch (e.g., `feature/awesome-feature`).
- **Make changes**: Implement your feature or fix in your branch.

## Code Style
- Follow the existing code style in the repository. 
- Use meaningful variable names and comments to make the code understandable.
- Ensure your code passes all relevant tests.

## Pull Request Process
1. **Open a pull request**: Once you're ready, navigate to the pull requests section of the repository and create a new pull request.
2. **Describe your changes**: Provide a clear description of what you've done and why. 
3. **Review process**: Your pull request will be reviewed by the maintainers. Be responsive to feedback and make necessary changes.

## Issue Reporting
- Before opening an issue, check if it already exists.
- Provide a descriptive title and a detailed description of the problem.
- Include steps to reproduce if applicable and any relevant logs or screenshots.

Thank you for contributing to BharatAI!
